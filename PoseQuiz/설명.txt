- showImage.py : 포즈 분석 결과를 skeleton 좌표가 찍힌 이미지로 출력하는 코드
- showAccuracy.py : 포즈 분석 결과를 수치화하여 두 player의 승/패 결과를 출력하는 코드


#showAccuracy.py
1. 라즈베리파이 카메라 모듈을 활용하여 실시간 이미지 촬영
 - 두 명의 player를 기준으로 함
 
2. 이미지 용량 축소
 - 카카오 Pose API를 활용하기 위한 전제 조건
 
3. 포즈 유사도 분석 알고리즘(수학 연산)
 1) Skeleton 분석 사진에서 기준이 되는 사진의 각 관절(17개의 점)의 x, y좌표를 이용하여 인접한 점들간의 기울기(tan값)를 구한다
 2) 기준이 되는 사진(이하 정답사진)과 입력받은 사진(이하 입력사진)의 기울기 값을 비교하여 오차를 구하고 오차들의 평균을 구한다.
    (입력 사진에는 사용자 2명이 인식되고 각각의 skeleton을 추출한다)
 3) 즉, 입력사진의 두 사람의 skeleton을 이용해 정답사진의 skeleton과의 각각의 오차율을 구한다
 
4. 포즈 분석 수행
 - 카카오 Pose API 활용 (RestAPI 수신)
 - 정답 이미지의 골격 keypoints 데이터값을 리스트에 미리 저장함
 (시연용 5개 포즈(ex.자유의여신상, 스카이콩콩...)에 대한 골격 keypoints 데이터값을 미리 저장할 예정)
 - 포즈 유사도 분석 알고리즘을 적용하여 두 player의 포즈 오차율 계산
 - 오차율을 바탕으로 승자/패자 결정
